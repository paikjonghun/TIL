## 26강 - 자바 열거 타입

핵심 키워드 : 열거 타입, 열거 타입 선언, 열거 상수, 열거 타입 변수

핵심 포인트 : 데이터 중에는 몇 가지로 한정된 값만을 갖는 경우가 있다. 이러한 한정된 값을 갖는 타입을 열거 타입이라고 한다. 

- 열거 타입
    - 열거 상수(한정된 값)를 저장하는 타입
        - week today;
        - today = Week.FRIDAY;
            - 한정된 값만 저장할 수 있음.
    - 열거 타입 선언
        - 1. 소스파일(.java) 생성
        - 2. 열거타입 선언
            - public enum 열거타입이름 { … }
        - 3. 열거 상수 선언
        - 4. 열거 상수는 열거 객체로 생성
        - 5. 열거 타입 변수 선언
        - 6. 열거 상수 저장
            - 열거타입 변수 = 열거타입.열거상수;
