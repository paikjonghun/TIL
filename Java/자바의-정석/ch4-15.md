## 중첩 for문

- for문 내에 또 다른 for문을 포함시킬 수 있다.
    
    ```java
    // 구구단 2단~9단 출력
    for(int i = 2; i <= 9; i++) {
    	for(int j = 1; j <= 9; j++) {
    		System.out.println(i+"*"+j+"="+(i*j));
    	}
    }
    ```
    

- 별 찍기
    
    ```java
    for(int i = 1; i <= 10; i++) {
    	for(int j = 1; j <= 10; j++) {
    		System.out.print("*");
    	}
    	System.out.println();
    }
    ```
