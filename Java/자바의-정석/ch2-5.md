# 상수와 리터럴

## 변수, 상수, 리터럴

- 변수(variable) - 하나의 값을 저장하기 위한 공간
- 상수(constant) - 한 번만 값을 저장 가능한 변수
    - `final int MAX = 100;`
- 리터럴(literal) - 그 자체로 값을 의미하는 것

```java
public class VarEx3 {
	public static void main(String[] args) {

		final int scroe = 100;
		score = 200; // 컴파일 에러 발생. score는 상수이기 때문
	}
}
```
