## for문

- 조건을 만족하는 동안 블럭{}을 반복 - 반복 횟수를 알 때 적합
    
    ```java
    for(초기화; 조건식; 증감식) {
    	수행될 문장
    }
    ```
    
    1. 초기화
    2. 조건식
    3. 수행될 문장 수행
    4. 증감식 → 조건식으로 이동

```java
for(int i = 1; i <= 5; i++) {
	System.out.println("I can do it.");
}
```

- 반복문에서 조건식은 주의해서 작성해야함 - 무한 반복할 수 있음.
- scope(범위) - 선언 위치부터 선언된 블럭의 끝까지
- 조건식을 생략하면, true로 간주되어서 무한반복문이 됨.
